Déclaration de conformité

Introduction

Cette page n'est pas une page d'aide, mais une déclaration de conformité au RGAA 3.0 qui vise à définir le niveau d'accessibilité général constaté sur le site conformément à la règlementation. Cette page est obligatoire pour être conforme au RGAA 3.0.

La déclaration de conformité du site Hal.archives-ouvertes.fr a été établie le vendredi 16 nov. 2018.

La version du RGAA utilisée pour réaliser les tests est la version 3.0 RC2

Technologies utilisées pour la réalisation du site

  • HTML5
  • CSS
  • JavaScript
  • Php

Mesure prise pour l'accessibilité

Pour la consultation

Des tags de la technologie Aria ont été positionné pour améliorer l'accessibilité

Pour le dépôt

Nous n'avons pas encore de test sur l'accessibilité ou la non accessibilité du dépôt.

Niveau d'accessibilité

Un diagnostic complet n'a pas encore été effectué sur le site Hal.archives-ouvertes.fr

Nous ne pouvons donc pas nous réclamer d'un niveau d'accessibilité.

Dérogations

Pages comportant une ou des dérogations

Page comportant une vidéo ou un piste audio

Les vidéos présentes sur le site n'ont en général pas d'audiodescription bien qu'elle en nécessiterait une. Ces vidéos n'étant pas la propriété du déclarant, il n'est pas possible d'intervenir ou d'en faire une modification au regard du respect du droit d'auteur.

De la meme facon, les fichiers audio n'ont pas de transcription

Notice comportant des fichiers Images

Les images présentes sur le site comme contenu scientifique, n'ont en général pas de description au dela de celle indiquée par le déposant. Ces vidéos n'étant pas la propriété du déclarant, il n'est pas possible d'intervenir ou d'en faire une modification au regard du respect du droit d'auteur.

Les fichiers Pdf, Doc,...

Les fichiers textes déposés sur le site ne sont pas, pour la majorité, accessibles. N'étant pas propriétaire de ces documents, il nous est impossible d'en assurer un niveau d'accessibilité minimum.

Amélioration et contact

Vous pouvez nous aider à améliorer l'accessibilité du site en nous signalant les problèmes éventuels que vous rencontrez. Pour ce faire, envoyez-nous un courriel à accessibilite@archives-ouvertes.fr.
